# Stack do Projeto

- **Tipo:** Backend - Monolito Modular
- **Linguagem:** Go (Golang) 1.22+
- **Arquitetura:** Hexagonal (Ports & Adapters)
- **Banco de Dados:** CockroachDB
- **Mensageria:** RabbitMQ
- **Router HTTP:** go-chi
- **Testes:** Testify (assert, require, mock) + Mockery
- **Observabilidade:** OpenTelemetry (Traces, Logs, Métricas)
- **Configuração:** Cobra + Viper
